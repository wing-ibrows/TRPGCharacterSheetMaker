
### 概要

* クトゥルフ第6版の探索者キャラクターシートの作成支援用のHTMLです。

* 記入管理できる情報：
  * 探索者情報
  * 能力値および能力値ロール
  * 探索者の技能
  * 探索者データ

* 提供する主要機能：
  * シート作成
  * 印刷
  * ファイルダウンロード
    * 各情報をjson形式のファイルでDLできます。
  * ファイル読み込み
    * 一度ダウンロードした情報を、後から再度読み込む事が可能です。

* その他特徴：
  * 「ロール」を押下すると、サイコロを一瞬で振ったかのように、自動的に各能力値が乱数で決定・投入されるようになっています。
  * 探索者の技能の加算計%、残加算%が自動計算・反映されるようになっています。
  * 投入情報に著しい不足があると、画面上部に赤文字で案内テキストが表示されます。

### 使い方

* 本プロジェクトをダウンロードし、```index.html```をブラウザで開くと利用できます。

### 注意事項

* 職業ごとの加算可能技能については、ご自身で調べたうえで投入ください。
  * 例外的な職業や補正などの可能性も加味して、職業欄は単なるテキストボックスで、かつ「職業加算%」も技能の入力制限を設けず自由に投入可能にしています。

* ファイル読み込み時の、バリデーション処理は実装していません。
  * 入力情報に```,```や```[]```、```{}```等の文字は利用しないようお願いします。

* デザインは最低限なのでご容赦ください。
  * シンプルなjqueryのみを利用し、bootstrapやその他ライブラリは利用していません。
